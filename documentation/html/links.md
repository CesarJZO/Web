# Links

## Anchor

If the `a` element has an `href` attribute, then it represents a hyperlink (a hypertext anchor) labeled by its contents

```html
<a href="path/or/url">Content</a>
```

Also, a link can be opened in a new tab

```html
<a href="path/or/url" target="_blank">New tab</a>
```

Linking to an element on the same page or to get back to the top of the page

```html
<a href="#section">Jump to the heading</a>
<a href="#top">Go back to top</a>
<a href="#">The same but writing less</a>
```

```html
<a href="#something">Go to something</a>
<!-- blah blah blah -->
<h3 id="something">I'm a header</h3>
```

It can be used to link to an email or a telephone number

```html
<a href="mailto:cesarjzo@outlook.com">Email</a>
<a href="tel:+123456789">Phone</a>
```
