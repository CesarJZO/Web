# Text

## Heading

```html
<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<!-- ... -->
<h6>Heading 6</h6>
```

## Paragraph

```html
<p> Represents a paragraph </p>
```

## Line break

```html
This is a line <br> and this is another line.
```

## Emphasis

```html
I want to <em>emphasize</em> something
```

## Strong importance

```html
This is <strong>important</strong>
```
